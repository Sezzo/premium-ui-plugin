# Example: Dashboard Redesign Workflow

This example demonstrates how to use the Premium UI Plugin to redesign an existing dashboard screen.

## Scenario

You have a SaaS analytics dashboard that looks generic and lacks visual hierarchy. You want to elevate it to premium quality using a systematic design approach.

## Step-by-Step Workflow

### 1. Audit the Current State

Use `design-system-review` to assess what exists:

```
Review my current design system. Focus on the dashboard components:
cards, charts, navigation sidebar, and data tables.
```

### 2. Define Token Foundations

Use `token-architecture` to establish proper design tokens:

```
Define a token architecture for my analytics product.
I need tokens for: colors (with semantic naming), typography scale,
spacing scale, border radius, and elevation/shadow levels.
```

### 3. Redesign the Screen

Use `screen-composition-design` to restructure the layout:

```
Redesign my analytics dashboard screen. It currently has:
- A sidebar navigation
- A top metrics row with 4 KPI cards
- Two chart sections (line chart, bar chart)
- A data table below

Make it feel premium and professional, not like a generic template.
```

### 4. Refine to Premium Quality

Use `premium-ui-refinement` to polish the result:

```
Refine this dashboard design to premium quality.
Focus on spacing consistency, typography hierarchy,
and making the data visualizations feel sophisticated.
```

### 5. Verify Consistency

Use `ux-consistency-audit` to check coherence:

```
Audit the redesigned dashboard for UX consistency.
Check navigation patterns, interaction states, and
whether the design language is coherent across all sections.
```

### 6. Get Critical Feedback

Use `design-critique` for a final quality check:

```
Critique this dashboard design. Be strict.
Does it feel premium? Is the hierarchy clear?
Would a serious B2B customer trust this product?
```

## Expected Outcome

After completing this workflow, you should have:

- A clear token system driving all visual decisions
- A dashboard layout with strong visual hierarchy
- Consistent spacing, typography, and color usage
- No generic template patterns or cheap visual effects
- A design that communicates professionalism and trustworthiness
