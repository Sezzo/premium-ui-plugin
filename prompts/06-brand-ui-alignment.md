# Prompt 06: Brand UI Alignment — Healthcare App

## Prompt

```
Check the brand-UI alignment for a healthcare patient portal using the brand-ui-alignment skill.

Brand values: Trust, clarity, calm, professionalism, empathy
Brand personality: Reassuring but not patronizing, clinical but not cold

Current UI:
- Primary color: bright orange (#FF6B00)
- Secondary color: electric blue (#0066FF)
- Font: Poppins (rounded, geometric)
- Icons: Outlined style with playful rounded corners
- Cards have large border-radius (16px) and colorful shadows
- CTAs use exclamation marks ("Book Now!" "Get Started!")
- Empty states use cartoon illustrations of doctors
```

## Expected Output Should Include

- **Brand-UI gap analysis** for each brand value
- **Color mismatch**: Bright orange contradicts calm/trust — healthcare needs muted, reassuring tones
- **Typography concern**: Poppins is friendly but may feel too casual for clinical trust
- **Tone issues**: Exclamation marks and "Get Started!" contradict reassuring professionalism
- **Illustration critique**: Cartoon doctors may undermine clinical credibility
- **Specific alternatives** with reasoning tied to brand values
- **Priority matrix** of changes

## Red Flags (Plugin Not Working If...)

- Accepts bright orange as appropriate for healthcare trust
- No connection between brand values and specific UI elements
- Generic advice not tied to the healthcare context
