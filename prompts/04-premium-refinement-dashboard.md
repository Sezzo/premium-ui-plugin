# Prompt 04: Premium UI Refinement — Analytics Dashboard

## Prompt

```
Refine this analytics dashboard to feel more premium using the premium-ui-refinement skill.

Current state:
- Header with app logo, navigation tabs, and user avatar
- Left sidebar with menu items using default system font
- Main content area with 4 metric cards in a row (white cards, thin gray border, 4px radius)
- A large line chart below the cards with a basic grid and default chart colors
- A data table at the bottom with alternating row colors (white/light gray)
- Everything uses 12px font size with no variation
- Spacing feels cramped — 8px gaps everywhere
- No hover states on interactive elements
```

## Expected Output Should Include

- **Before/after analysis** for each element
- **Typography refinement**: Introduce hierarchy (not just one font size)
- **Spacing improvements**: Specific token-based spacing recommendations
- **Card elevation**: Shadow and radius adjustments with token references
- **Chart refinement**: Color palette, grid styling, label typography
- **Table improvements**: Row height, alignment, hover states
- **Interaction states**: Hover, focus, active for all interactive elements
- **Specific token references** for every recommendation

## Red Flags (Plugin Not Working If...)

- Suggests adding gradients, glows, or decorative effects
- Recommendations are vague ("make it look better")
- No token references
- Suggests emojis or playful elements for a data dashboard
