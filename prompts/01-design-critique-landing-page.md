# Prompt 01: Design Critique — Landing Page

## Prompt

```
Review this landing page design critically using the design-critique skill:

The landing page has:
- A hero section with a large gradient background (purple to blue), white headline text "Welcome to the Future of AI", a sparkle emoji in the headline, and a glowing CTA button
- A features section with 3 cards, each with a robot-head icon, rounded corners (20px), and drop shadows
- A testimonial carousel with auto-play and dot navigation
- A footer with social media icons and a newsletter signup

The target audience is enterprise B2B buyers for a data analytics platform.
```

## Expected Output Should Include

- **Overall rating** (likely low, 3-5/10 range)
- **Critique of AI clichés**: The sparkle emoji, robot-head icons, and "Future of AI" headline should be flagged as forbidden patterns
- **Gradient criticism**: Decorative gradient without purpose should be called out
- **Glow effects**: Glowing CTA button flagged as fake-premium
- **Typography assessment**: Evaluation of headline hierarchy
- **B2B audience mismatch**: Enterprise buyers expect serious, trustworthy design — not trendy startup aesthetics
- **Specific recommendations** with token references
- **Priority action table** with effort estimates

## Red Flags (Plugin Not Working If...)

- Output praises the gradient or glow effects
- No mention of forbidden patterns (emojis, robot icons, AI clichés)
- Generic feedback without specific token or system references
- No structured critique format (just paragraphs of vague advice)
