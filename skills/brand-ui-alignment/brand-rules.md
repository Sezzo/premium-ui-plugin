# Brand UI Alignment Rules

## Core Principle
Brand alignment in UI is not decoration.
It is the consistent expression of product character through system decisions.

## Evaluate these dimensions

### 1. Brand Character
Clarify the intended brand attributes.
Examples:
- calm
- intelligent
- premium
- precise
- trustworthy
- warm
- technical
- discreet
- bold

The UI should express a coherent subset, not all possible adjectives at once.

### 2. Typography Character
- Does the typography support the intended tone?
- Does it feel serious, calm, confident, and mature where needed?

### 3. Color Character
- Do colors reinforce the intended identity?
- Do they feel ownable and disciplined?
- Are they too generic, too loud, or too trend-dependent?

### 4. Component Personality
- Do components reflect the same product character?
- Or do they mix conflicting moods and styles?

### 5. Motion Character
- Does motion feel aligned with the brand?
- Is it restrained and credible?
- Or playful, flashy, or theatrical without reason?

### 6. Trust and Seriousness
- Does the interface support trust?
- Does it look credible for a premium product?
- Does anything feel cheap or unserious?

### 7. Distinctiveness
- Could this belong to any random startup?
- Or does it feel recognizably tied to one product?

## Red Flags
- generic SaaS identity
- copied aesthetics from major brands
- trendy but anonymous styling
- trying to look premium through effects
- inconsistent tone across screens
- emoji-like UI accents
- AI-themed gimmicks
- decorative iconography with no brand logic

## System-Level Brand Rules
Strong brand alignment usually comes from:
- a disciplined type system
- restrained and characterful color usage
- consistent spacing and density
- component shapes and surfaces with clear logic
- motion that matches product tone
- a stable hierarchy model
- a serious and repeatable visual language

## Final Questions
- Does this UI actually express the intended brand?
- Does it feel ownable?
- Does it support trust?
- Does it avoid cliché and imitation?
- Can the same brand logic scale across future screens and components?
