# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2025-03-25

### Added

- Initial release of Premium UI Plugin
- 10 specialized design-system skills:
  - `design-system-setup` — Bootstrap repository with premium UI standards
  - `design-system-review` — Review design system consistency and completeness
  - `ui-component-design` — Design reusable UI components with proper states and variants
  - `premium-ui-refinement` — Elevate existing UI to premium quality
  - `ux-consistency-audit` — Check cross-screen UX coherence
  - `screen-composition-design` — Build or restructure full screens
  - `token-architecture` — Define scalable token systems
  - `component-implementation-spec` — Create implementation-ready engineering specs
  - `design-critique` — Critical review of design work
  - `brand-ui-alignment` — Check UI brand expression
- Plugin manifest (`plugin.json`)
- Recommended workflows for common design tasks
- Forbidden patterns list to prevent generic/cheap UI
- Comprehensive README with installation and usage instructions
